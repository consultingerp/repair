Fleet Repair Management

This module is useful for Fleet Repair Management. Fleet is generalized for any equipment here. Based on information provided for any Fleet that needs repairing,generated diagnosis and work order. System will create quotations,sale Orders, invoice and payments for the whole process done on the Fleet.

Features

Create fleet/car repair order.

You can create car repair order with car related information.

Add fleet/car service details.

You can add fleet/car service details like (Car guarantee,Guarantee type).

Fully Automated workflow.

f you want to Group By some data with specific fields, Now it's possible in excel reports. You can Group By selected fields.

Create fleet/car diagnosis.

You can create diagnosis based on fleet repair order.

Manage fleet/car workorder.

You can manage fleet/car repair work order (start,pending,resume,cancel).

Assign to Technician and Print Report..

You can assign fleet/car diagnosis to technician and print fleet repair,dignosis, workorder reports.

Access Rights and Roles

We have four different access groups for machine repair module.

  • Directeur Commercial
  • Service Manager
  • Head Technician
  • Technician

Roles:


  • Technician : Technician can work on work order.He/She can only see his/her work order and only work on his/her assigned work order.Technician can only have a read access to diagnosis and repair order.
  • Head Technician : Head technician can see all work order also able to assign a technician on a dignosis and work order. he/she has a access to enter a dignosis result. He/She can not chnage any thing on a repair order.
  • Service Manager : Service Manager ceates repair order.convert it as a dignosis able to create quotations.
  • Directeur Commercial : Directeur Commercial is a admin of this system he/she has full access rights of every thing.
